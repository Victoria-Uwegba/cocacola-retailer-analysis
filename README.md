# Coca-Cola USA Retail Sales Dashboard

![Dashboard Preview](./COCA%20COLA%20DASH%20BOARD.PNG)

## Overview
A comprehensive retail performance dashboard tracking Coca-Cola's US sales operations, providing insights into:
- Regional sales performance
- Year-over-year growth analysis
- Product category performance
- Retailer partnership effectiveness

## Key Features
### Real-time Metrics
- Total Sales: \[$12,016,665\]
- Units Sold: \[9,648\] units
- Average Price/Unit: \[$4,362.5\]
- Total Profit: \[$4,722,497\]

### Interactive Visualizations
1. **Brand Performance Comparison**
   - Year-over-year sales growth (2022 vs 2023)
   - Variance analysis for key brands:
     - Coca-Cola: \[+1,769,873\]
     - Dasani Water: \[+1,448,557\]
     - Sprite: \[+987,000\]

2. **Quarterly Sales Analysis**
   - Dual-axis chart showing:
     - Quarterly sales figures
     - Operating profit margin trends

3. **Regional Sales Breakdown**
   - Geographic performance across:
     - Midwest
     - Northeast
     - South
     - Southeast
     - West

### Retailer Performance
- Top performing retailers analysis:
  - Amazon: \[+733,489\] growth
  - BevCo: \[+1,860,819\] growth
  - Walmart: \[+240,299\] growth

## Technology Stack
- **Frontend**: React + TypeScript
- **Data Visualization**: Recharts
- **Styling**: Tailwind CSS
- **Data Processing**: Python/Pandas (ETL pipelines)
